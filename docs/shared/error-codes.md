# 에러 코드

## 공통 에러

| 코드 | HTTP Status | 설명 | 대응 방법 |
|------|-------------|------|-----------|
| `BAD_REQUEST` | 400 | 잘못된 요청 파라미터 | 요청 값 검증 후 재시도 |
| `VALIDATION_ERROR` | 400 | 입력값 유효성 검증 실패 | message 필드에서 어떤 필드가 잘못됐는지 확인 |
| `UNAUTHORIZED` | 401 | 인증 실패 | 로그인 페이지로 리다이렉트 |
| `NOT_FOUND` | 404 | 리소스를 찾을 수 없음 | 요청 경로/ID 확인 |
| `INTERNAL_ERROR` | 500 | 서버 내부 오류 | 재시도 후 지속되면 문의 |

## Auth 도메인

| 코드 | 발생 조건 | 프론트 처리 |
|------|-----------|-------------|
| `UNAUTHORIZED` | 이메일/비밀번호 불일치 | "이메일 또는 비밀번호를 확인해주세요" 표시 |
| `BAD_REQUEST` | 이미 가입된 이메일로 회원가입 시도 | "이미 가입된 이메일입니다" 표시 + 로그인 유도 |
| `VALIDATION_ERROR` | 비밀번호 8자 미만 등 | 각 필드별 에러 메시지 표시 |

## Member 도메인

| 코드 | 발생 조건 | 프론트 처리 |
|------|-----------|-------------|
| `NOT_FOUND` | 존재하지 않는 회원 ID | 로그아웃 처리 + 재로그인 유도 |
